---

## 5) CSS Import Order Fix

### 5.1 Issue

CSS variables were not applied correctly due to incorrect import order in `main.jsx`.

### 5.2 Solution

Updated import order:

1. Bootstrap CSS
2. `design-tokens.css`
3. `common.css`
4. `index.css`
5. `App.css`

This ensures CSS variables are defined before consumption.
