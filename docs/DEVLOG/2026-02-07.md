
# Project PFE – Day 1 Setup Log

## Date: 07/02/2026

## Goal
Initialize the development environment (monorepo) and confirm that the 3 main services run correctly: **Laravel backend**, **React frontend**, **FastAPI AI service**.

---

## 1) Project Structure
Created a monorepo structure with:
- `backend/` → Laravel API + MySQL
- `frontend/` → React (Vite) + Bootstrap later
- `ai/` → FastAPI service (OCR/extraction to be implemented)
- `docs/` → documentation, diagrams, report assets

---

## 2) AI Service Setup (FastAPI)

### Environment
- Created virtual environment: `.venv`
- Installed packages:
  - `fastapi`, `uvicorn[standard]`, `python-multipart`
  - (extra for testing): `numpy`, `pandas`, `scikit-learn`
- Created `main.py` with endpoints:
  - `GET /health`
  - `POST /process` (dummy response for now)
- Generated `requirements.txt`:
  ```bash
  pip freeze > requirements.txt
  ```
- Added `ai/.gitignore` to ignore:
  - `.venv/`, `__pycache__/`, `*.pyc`, `.env`

### Run / Test
```bash
cd ai
.\.venv\Scripts\activate
python -m uvicorn main:app --reload --port 8001
```

Checks:
- Visit `http://127.0.0.1:8001/health` → should return:
  ```json
  {"status":"ok"}
  ```
- Swagger UI available at: `http://127.0.0.1:8001/docs`
