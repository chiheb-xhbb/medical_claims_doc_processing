## 3) Database â€“ Dossier Persistence Layer

### 3.1 New table: `dossiers`

Created `dossiers` table with:

- `numero_dossier` (string, unique)  
- `assured_identifier` (string(8), indexed)  
- `status` (string, indexed, default `RECU`)  
- `montant_total` (decimal(10,3), default `0.000`, indexed)  
- optional fields: `episode_description`, `notes`  
- audit fields:
  - `created_by` (FK users, `nullOnDelete`)
  - `submitted_at` (timestamp nullable)
  - `validated_by` (FK users, `nullOnDelete`)
  - `validated_at` (timestamp nullable)
- timestamps

**Domain naming decision:** `assured_identifier` was chosen (instead of `patient_identifier`) to match insurance domain reality: reimbursement is attached to the **insured person**.

---

### 3.2 Linking documents to dossiers

Added nullable FK on `documents`:

- `documents.dossier_id` â†’ `dossiers.id` (`nullOnDelete()`)

**Reasoning:** documents and their AI extraction history remain valuable artifacts even if a dossier is deleted. This prevents destructive cascades and preserves traceability.

## 5) Domain Rules â€“ Total & Lifecycle Controls

### 5.1 Frozen total rule
Implemented:

- `isTotalFrozen()`  
  Total becomes immutable when dossier reaches final states:
  - `VALIDE`, `REJETE`, `EXPORTE`

### 5.2 Deletion rule
Implemented:

- `canBeDeleted()`  
  Prevent deletion for:
  - `VALIDE`, `EXPORTE`

### 5.3 Total calculation (source of truth = versioned extraction JSON)
**Context:** `documents` table does not store `total_ttc`.  
The monetary value is stored inside `extractions.result_json['fields']['total_ttc']`, and corrections are audit-tracked via extraction versioning + `field_corrections`.

MVP implementation:

- `getCurrentTotal()`:
  - selects dossier documents with `status = VALIDATED`
  - for each document, loads the latest extraction version (`latest('version')`)
  - sums `result_json.fields.total_ttc`

- `getDisplayTotal()`:
  - returns stored `montant_total` when frozen
  - otherwise returns dynamic computed total

- `updateTotal()`:
  - updates `montant_total` only when not frozen (uses `saveQuietly()`)
